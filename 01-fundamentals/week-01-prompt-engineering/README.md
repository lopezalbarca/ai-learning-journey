# Fundamentals - 01 - Prompt Engineering

## 🧠 Goal

This week is focused on learning the fundamentals of prompt engineering for Large Language Models (LLMs), with hands-on practice using the OpenAI Playground and/or the official .NET SDK.

By the end of the week, I aim to:
- Understand the key prompt engineering techniques (zero-shot, few-shot, role prompting, format instructions, etc.)
- Practice writing and refining prompts for common use cases
- Create a personal prompt library (`prompt-pack.md`) with reusable patterns
- Begin building intuition for how LLMs interpret and respond to input
