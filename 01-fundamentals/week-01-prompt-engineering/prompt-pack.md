# 🧠 Prompt Pack – Week 1

A curated collection of prompts tested during my first week of learning AI prompt engineering.

## ✨ Prompt Examples

### 🗣️ Role-based Prompt
> _"You are a sarcastic senior developer mentoring junior devs. Explain what 'asynchronous' means."_

### 🛠️ Tool Prompt
> _"Summarize this meeting transcript in 3 bullet points using Markdown."_

### 📊 Data Prompt
> _"Turn the following text into a CSV format with columns: Name, Role, Skills."_

## 📝 Notes

- Prompts that include **explicit output format** (like Markdown or JSON) tend to be more stable.
- Giving a **role** to the model often improves coherence.
- Iterative refinement (like "Try again with more details") is useful when used programmatically.
